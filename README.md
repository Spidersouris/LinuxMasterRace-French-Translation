## Vous pouvez choisir n'importe quelle version de GNU / Linux que vous voulez

Contrairement à des systèmes d'exploitation traditionnels, il n'y a pas un système d'exploitation appelé "GNU / Linux". A la place, il y a beaucoup de différents systèmes d'exploitation basés sur le noyau Linux et les paquets GNU, appelés «Distributions» ou «Distros» pour faire plus court. Cela signifie que si vous n'aimez pas la façon dont un distro fonctionne, vous pouvez choisir un autre et toujours profiter de tous les avantages de GNU / Linux. Si aucune ne vous conviennent, et - surtout - êtes suffisamment qualifiés, vous pouvez même créer votre propre distro!

## GNU / Linux ne coûte rien à utiliser

Vous pensez probablement : "Oh, je n'ais pas payé pour Windows". Êtes-vous absolument sûr? Si votre ordinateur est livré avec une copie de Windows, alors vous l'avez payé, même si le vendeur ne vous l'a pas dit. Le prix d'une licence Windows équivaut en moyenne au quart du prix de chaque nouvel ordinateur. Donc, à moins que vous piratiez Windows, vous l'avez probablement payé. Où pensez-vous que Microsoft tire son argent?
De l'autre côté, vous pouvez obtenir GNU / Linux entièrement gratuitement. C'est ça, tous ces gars-là dans le monde entier ont travaillé très dur pour faire un OS soigné, sûr, efficace et beau, et ils donnent leur produit à tout le monde pour l'utiliser librement. Bien sûr, certaines entreprises font de bonnes affaires en vendant support, documentation, hotline, etc., pour leur propre version de GNU / Linux, mais la plupart du temps, vous n'aurez pas besoin de payer un centime.

## GNU / Linux peut fonctionner sur tout système sans latences ou ralentissements

Windows nécessite de plus en plus de puissance matérielle au fur et a mesure que son numéro de version augmente (XP, Vista, 7, 8, 10, etc.). Donc, si vous voulez rester sous Windows, vous devez constamment obtenir du nouveau matériel. Non seulement cela est inconvénient, mais cela prend aussi votre argent et la donne directement à Microsoft.

GNU / Linux est différent. GNU / Linux est livré avec de nombreux distros, environnements de bureau et applications légers, afin de garder votre ancien ordinateur en état de marche. Bien sûr, il ne ferez pas une course-gagnant de votre ordinateur portable de 14 ans, mais ça va être suffisant pour effectuer des tâches de base dont la plupart des utilisateurs finaux sont rarement épargnés, comme la navigation Web, les e-mails et le téléchargement de petits fichiers. Jusqu'à ce que quelqu'un peut obtenir Windows 95 fonctionnant sur une plate-forme Windows 7, GNU / Linux sera toujours le roi de l'héritage.

## Dites au revoir à l'installation des pilotes

Les nouveaux périphériques, même les plus simples, viennent généralement avec un CD. Sur le CD, un très petit logiciel appelé un "driver". Si vous lisez le manuel d'instructions, vous saurez que le périphérique ne fonctionne pas sur un ordinateur Windows jusqu'à ce que vous installez le pilote. Si vous ne lisez pas le manuel, alors vous aurez probablement le découvrir vous-même quand vous voyez votre nouvelle installation ne fonctionne pas hors de la boîte.
Insérez le CD, cliquez sur l'assistant d'installation, attendez, éjecter le CD, l'ordinateur de redémarrer.
Si vous avez acheté le périphérique il y a un certain temps et réutilisez sur un autre ordinateur, ou si vous avez réalisé combien obsolètes disques optiques sont et ne disposez pas d'un lecteur de disque optique sur votre ordinateur, vous aurez probablement envie d'oublier le CD et de récupérer la dernière version du pilote à partir du site Web du fabricant. Qui peut prendre un peu de temps, étant donné l'organisation étrange de certains sites Web de certains fabricants.
Bon, donc ça c'est pour cet unique périphérique. Maintenant, imaginez que vous souhaitez installer Windows sur un tout nouvel ordinateur. Pour chaque petit périphérique, vous devrez trouver le dernier pilote (ou utiliser un CD), l'installer et redémarrer de temps en temps. Carte vidéo, carte son, clavier, souris, chipset carte mère, etc. (mieux vaut faire le pilote de la carte vidéo en premier ou vous êtes coincé avec votre écran haut de gamme montrant un mode de résolution très faible). Et c'est ce qui vient après une déjà assez longue installation de Windows lui-même.

GNU / Linux n'a pas besoin de pilotes distincts. Tous les pilotes sont déjà inclus dans le noyau Linux, et cela vient avec chaque installation GNU / Linux unique.

## GNU / Linux respecte votre liberté

GNU / Linux est [Logiciel libre,](https://www.gnu.org/philosophy/free-sw.html) ce qui signifie qu'il respecte vos libertés d" l'utiliser, de le modifier et de le partager lui et tout ce que vous créez à l'aide GNU / Linux. Vous êtes libre d'utiliser GNU / Linux à des fins commerciales ou autres, sans aucune restriction.

D'autres systèmes d'exploitation ne permettent généralement pas de le modifier ou de les partager, ou même simplement regarder leur code source pour vous assurer qu'ils ne font rien de méchant. En outre, leurs fournisseurs peuvent restreindre ce que vous faites avec eux, par exemple le nombre d'appareils que vous pouvez connecter à votre ordinateur (Windows et OS X), si vous pouvez exécuter un serveur en utilisant votre ordinateur (Windows et OS X), si vous pouvez utiliser votre ordinateur à des fins commerciales (OS X et certaines licences Windows), si vous pouvez vous connecter à votre ordinateur à distance (Windows et OS X), ou si vous pouvez exécuter le système d'exploitation sur du matériel non approuvé (OS X).

Certains fournisseurs de systèmes d'exploitation, y compris Microsoft et Apple, ne permettent pas d'utiliser leurs systèmes d'exploitation pour enfreindre la loi. Alors, cela est parfaitement bien quand il vient à des lois justes (et vous avez encore à les suivre de toute façon), mais vous pouvez également être soumis à des lois injustes, comme les lois interdisant les manifestations, les actes de désobéissance civile, ou de critiquer le gouvernement, ou d'autre lois de censure, que vous devriez être en mesure de briser.

Certains fournisseurs de système d'exploitation, comme Microsoft, exigent également vous acceptez [arbitrage exécutoire](https://www.microsoft.com/en-us/legal/intellectualproperty/copyright/default.aspx) afin d'utiliser leurs systèmes d'exploitation . Cela est mauvais pour plusieurs raisons, mais surtout parce qu'il limite votre accès aux tribunaux et votre capacité à participer à des recours collectifs.

Votre ordinateur ne doit pas être en mesure de vous dire ce que vous pouvez et ne pouvez pas faire.

Et si vous utilisez GNU / Linux, il n'en sera pas.

Non seulement cela, mais [la NSA a essayé, et raté](http://alt.os.linux.ubuntu.narkive.com/KqeZha9Y/ot-linus-confirms-that-the-nsa-has-asked-him-to-put-a-backdoor-in-linux) de détourner les données de Linux, et même s'ils l'avait fait, [ce détournement serait pratiquement inutile.](http://tildehash.com/?blog=the-nsa-backdoor-in-linux)

## Aucun Malware

Cela pour 4 grandes raisons

1. Quand quelqu'un fait un Malware, ils veulent qu'il affecte la plupart des gens et être en mesure de se propager rapidement. Quel système d'exploitation ne la plupart des gens utilisent? Windows. Plus de 11.000 virus uniques ont été faites pour Windows (source: Panda Labs). Parce que seulement 16% des logiciels malveillants se compose de virus (source: Panda Labs), par simple extrapolation, nous pouvons dire qu'il ya 68,750 pièces différentes de logiciels malveillants qui peuvent affecter un système Windows en ce moment! Maintenant, nous allons comparer cela à GNU / Linux. Selon le wiki d'Ubuntu, il y a seulement 35 logiciels malveillants qui ait jamais affecté Linux (sur 89 tentatives), seulement 17 qui ont réussi à faire du mal au système qu'ils étaient quand ils ont été faits, seulement 8 d'entre eux ont réussit à se répendre "dans la nature", et exactement 0 peut effectivement faire de mal à un système moderne GNU/Linux.

2. Comme indiqué ci-dessus, tout le monde peut regarder et soumettre des modifications au système GNU / Linux. Ceci, combiné avec suivi des bogues de la communauté et des outils de collaboration, signifie que les bugs et les problèmes de sécurité sont résolus à une immense vitesse, habituellement dans les 1 ou 2 semaines pour les bugs à faible risque, 2 ou 3 jours pour les bugs à haut risque. Dans un cas célèbre, un bug dans Ubuntu que signifiait les gens pouvaient se connecter à un utilisateur sans connaître le mot de passe en manipulant le clavier à l'écran, la facilité d'accès et les touches de fonction, ont été fixé en moins de 9 heures. Mais il n'y a pas que la pure vitesse a laquelle les bugs sont fixée. Dites que vous avez trouvé un bogue dans Windows (comme vous le faites). Qu'est-ce que vous feriez? Vous pouvez l'ignorer, peut-être essayer de faire une solution de contournement temporaire? Si vous étiez préoccupé assez, vous pouvez le signaler à Microsoft et nous espérons qu'ils fixent en quelques mois, ou dans la prochaine version de Windows, ou celle d'après. Dans Linux, vous pouvez l'afficher sur un bug tracker, et, si vous êtes si enclin à le faire, vous pouvez poster un patch vous-même.

3. Un virus peut se propager que si il peut affecter la même vulnérabilité dans chaque système, il atteint. Parce que vous n'êtes pas autorisé à modifier Windows ou OS X plus que Microsoft ou Apple vous le permettent, la plupart des versions de Windows et OS X sont presque identiques et contiendra les mêmes vulnérabilités. Comme indiqué plus haut, l'écosystème GNU / Linux est diverse, et chaque versions est unique, en raison des milliers de différentes personalisations disponibles, de sorte que les logiciels malveillants ne peuvent se propager aussi facilement.

4. Comme nous le verrons ci-dessous, presque tous les logiciels GNU / Linux sont géré dans un vaste répertoire, et pas téléchargé à travers le world wide web. Cela signifie que aucun malware peut être téléchargé (de la bonne façon) sur GNU / Linux. Et même si un Malware est obtenu dans le référentiel, il pourrait facilement être désherbé par quelqu'un et puis retiré.


## Accès et mise à jour tous vos logiciels en un seul clic

Si vous souhaitez installer un logiciel sous Windows, vous aurez besoin de:

1. Rechercher sur le Web pour trouver quel logiciel convient à vos besoins.

2. Trouver un site Web qui vous permet de le télécharger.

3. Peut-être le payer.

4. Téléchargez-le.

5. Installez-le.

6. Parfois, redémarrez votre ordinateur.

Ouf, ça fait beaucoup de travail juste pour essayer quelque chose de nouveau!
Avec GNU / Linux, tout est beaucoup plus simple. Les distros GNU / Linux ont ce qu'on appelle un «gestionnaire de paquets»: chaque logiciel est contenu dans son propre «paquet». Si vous avez besoin d'un nouveau logiciel, il suffit d'ouvrir le terminal, tapez une commande, entrez votre mot de passe et appuyez sur Y.
GNU / Linux dispose également d'une mise à jour automatique du logiciel. Donc, environ une ou deux fois par semaine, votre système d'exploitation vous demandera si vous souhaitez installer des mises à jour de votre logiciel (ou vous pouvez les définir pour être automatique) et, si vous cliquez sur Oui, tous vos logiciels seront mise à jour. Mais il y a plus! Vous avez déjà vu [ce](https://i.imgur.com/5UCpYVi.png) message avant? Si vous êtes un utilisateur Windows, vous avez probablement. Si vous êtes un utilisateur de GNU / Linux, le message ressemblera plus à [celui-ci](Https://i.imgur.com/O5Q7wDm.png).

Dans GNU / Linux, vous pouvez non seulement continuer à partir de là où vous vous êtes arrêté après une mise à jour, vous pouvez utiliser vos applications alors qu'ils sont en mise à jour.

## Pas de fragmentation

Si vous savez déjà ce que la fragmentation est, et êtes déjà habitués à faire la défragmentation de votre disque tous les mois, voici la version courte: GNU / Linux n'a pas besoin de défragmenter.

Si vous ne savez pas ce que fragmenter est, votre disque dur est divisé en petits morceaux de, disons, 1 mégaoctet. Dites que vous voulez installer foo, et à des fins de simplification, disons foo est 5 mégaoctets. Sous Windows, le disque dur est recherché pour des morceaux libres, et des morceaux de l'application sont mis dans le premier morceau vide, jusqu'à ce que l'ensemble de l'application est sur le disque dur. Non seulement cela rend l'application lente à utiliser et à lancer, mais cela rend aussi le processus d'installation plus long. La solution de Microsoft à cela était de faire une fonction de défragmentation pour trier le disque dur, et le mettre à l'état d'un disque dur GNU / Linux.

Sur GNU / Linux, le système conserve un enregistrement de morceaux vides contigus. Lorsqu'un nouveau fichier arrive, il recherche cette liste pour une assez longue rangée de morceaux vides, et place foo là. Cela signifie que, à la condition qu'il y a une rangée assez longue, votre disque dur est toujours propre, bien rangé et facilement accessible.

## Votre système est à vous

GNU / Linux supprime toutes les limites et l'abstraction entre vous et le système, en fournissant plus de contrôle, tout en simplifiant simultanément la maintenance du système. Un nouveau système GNU / Linux ne contient que des composants de base de base sans configuration automatique effectuée. Vous êtes en mesure de configurer votre ordinateur comme vous le souhaitez. Dès le début de la procédure d'installation, tous les composants de GNU / Linux sont 100% transparent et accessible pour un accès instantané, le retrait ou le remplacement par des composants alternatifs.

La nature ouverte de GNU / Linux implique également une courbe d'apprentissage assez raide, mais les utilisateurs de GNU / Linux expérimentés ont tendance à trouver d'autres systèmes plus fermés beaucoup plus gênant pour contrôler. Le principe de l'ouverture étend à ses membres de la communauté ainsi, car les utilisateurs de GNU / Linux sont très ouverts avec l'aide et la contribution.

Ce que cela signifie pour vous est que vous aurez un contrôle complet et la responsabilité de *votre* ordinateur. Windows et OS X cherchent à être convivial, mais GNU / Linux s'efforce d'être centrée sur l'utilisateur.

## Pas de crapware

Si vous avez déjà acheté un nouvel ordinateur exécutant Windows, vous devez savoir que cela peut être une expérience très frustrante.

Tout commence lorsque vous l'allumez: d'innombrables fenêtres commencent à ouvrir, vous demandant de souscrire à des services, la version premium d'un antivirus, les jeux que vous jamais demandé, des applications de productivité qui vous oblige à créer un compte en ligne, etc.

Mais ça ne s'arrête pas. Chaque fois que vous démarrez votre ordinateur, tous les programmes pré-installés doivent commencer dans l'arrière-plan, et il faut attendre plus longtemps et plus entre le moment où vous voyez votre bureau apparaissent sur l'écran et le moment où elle cesse d'être beaucoup trop lent à utiliser parce que tous ces programmes démarrent.

Pour couronner le tout, souvent après que vous possédez votre nouvel ordinateur pendant 30 jours, de nouvelles boîtes de dialogue commencent à apparaître, et vous rendre compte que certains de ces programmes que vous avez pensé étaient libres ne sont pas vraiment.

La chose est, la plupart des fabricants d'ordinateurs pensent que cela est une bonne chose. Ils installent des programmes sur votre ordinateur avant de l'acheter, pensant qu'ils vont améliorer votre expérience, parce que vous obtenez «plus» pour le même prix, et ils croient que cela peut les aider à différencier des concurrents, qui ne peuvent offrir que de nombreuses "améliorations" sur le système par défaut, ou peut-être pas les mêmes. Pour eux, c'est une valeur ajoutée.

Mais pour l'utilisateur, c'est surtout plus de douleur, plus d'attente, plus de désinstallation de programmes indésirables, plus d'argent pour payer si vous décidez que vous avez besoin de ce pas-si-libre-après-tout antivirus, et à la fin de la journée, un ordinateur gonflé et paralysé. Voilà pourquoi ces programmes ont été donné un nom pas très poli: "crapware".

Rien de tout cela se produit avec Linux. Sur GNU / Linux, il n'y a aucun logiciel sauf les bases (navigateur Web, Office Suite, E-Mail Client, etc.) installé sur un système de base, et certainement pas de logiciel d'essai!


## Le plus les gens utilisent GNU / Linux et le mieux il devient

Supposons un instant que cette année est enfin l'année du bureau GNU / Linux. Tout à coup, nous avons 10x le nombre d'utilisateurs. Avec cela, nous pouvons supposer, à juste titre, que nous avons 10x les contributeurs. LibreOffice est la nouvelle suite de fait office de standard, et tout le matériel et les logiciels sont désormais pris en charge sur GNU / Linux. Le logiciel est en cours de portage tous les jours, et les gens se rendent compte des avantages des logiciels open source.

Cela ne vaut pas seulement pour GNU / Linux, mais toutes les applications open source ainsi. Plus les gens qui utilisent une application, plus il obtient d'attention du public, donc plus de développeurs commencent à contribuer. En répartissant GNU / Linux, vous le rendez mieux pour tout le monde.

## Les changements controversés ne peut être forcé sur vous dans GNU / Linux

Avec de nombreux utilisateurs qui refusent de faire le saut à partir de Windows 7 à 8, il y a beaucoup de gens qui ne peuvent pas avoir un système d'exploitation à l'aise sur Windows 7 une fois la date d'expiration atteinte. Ce n'est pas un problème avec GNU / Linux. Vous aimez l'environnement Unity d'Ubuntu? Vous avez des options. MATE, XFCE, GNOME, Cinnamon et KDE ne font que gratter la surface, et avec leurs options de personnalisation étendues, vous avez règne plus libre que Windows ne pourrait jamais rêver. L'ensemble du système est à vous pour jouer avec. Personne ne peut forcer des changements drastiques sur vous tels qu'une interface orientée tactile. Personne ne peut vous faire utiliser un compte en ligne pour votre connexion PC. Personne ne peut vous forcer à mettre à jour votre système par le biais d'un magasin étroitement contrôlé. C'est le votre.
